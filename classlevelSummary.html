Detailed Description

A summary dialog window, which shows the exercise result.

Each value can be set using the functions.
They're all optional and unset values will be hidden.

The dialog supports 2 modes.
The first mode asks the user whether to continue to the next exercise or repeat it.
The second mode shows the summary with an OK button. It can be enabled by calling showOK().
