Detailed Description

The packView class is the core part of the pack editor.

packEditor uses this widget for tab content.
packView should be used only in packEditor (it communicates with its tab widget). This might change in future.
Creating a new file:

packView pack;
pack.openFile("Unnamed.typer",true,false) // tr() should be used for translation here

Opening existing file:

pack.openFile(fileName,false,false)

Opening existing file read-only:

pack.openFile(fileName,false,true)

packView saves opened files in the program configuration directory.
They're automatically removed after they're closed.
