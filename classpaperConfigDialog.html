Detailed Description

The paperConfigDialog class is a dialog, which asks for exercise configuration.

The dialog asks for line length and new line inclusion, but it might be extended with more
features in the future.

Line length can be accessed using the lineLength attribute.

paperConfigDialog myDialog;
myDialog.exec();
int length = myDialog.lineLength; // Line length
bool includeNewLines = myDialog.includeNewLines; // New line inclusion
